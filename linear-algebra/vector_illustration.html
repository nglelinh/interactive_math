🔢 Vector và Vector Space - Nền tảng Đại số Tuyến tính

📐 Vector Theory Foundations

Vector: Một đối tượng toán học có cả độ lớn và hướng, được biểu diễn như một mũi tên từ gốc tọa độ đến một điểm trong không gian.

Vector Space (R²): Tập hợp tất cả vectors 2 chiều với các phép toán cộng vector và nhân scalar.

Ý nghĩa hình học: Vectors mô tả chuyển động, lực, vận tốc, và các đại lượng vật lý có hướng.

🔍 Khái niệm cơ bản:

  • 🎯 Position Vector: Vector từ gốc tọa độ đến một điểm P(x,y)
  • 📏 Magnitude: Độ dài vector |v| = √(x² + y²)
  • 📐 Direction: Góc θ = arctan(y/x) với trục Ox
  • ⚖️ Unit Vector: Vector có độ dài bằng 1

📐 Dot Product và Geometric Interpretation

🔢 Mathematical Formula:

A·B = |A||B|cos(θ) = a₁b₁ + a₂b₂

Geometric Meaning: Dot product measures how much two vectors point in the same direction.

📊 Dot Product Values:

  • A·B > 0: Acute angle (θ < 90°) - vectors point in similar directions
  • A·B = 0: Right angle (θ = 90°) - vectors are orthogonal (perpendicular)
  • A·B < 0: Obtuse angle (θ > 90°) - vectors point in opposite directions

📏 Cosine Similarity:

cos(θ) = (A·B)/(|A||B|) - normalized dot product ranging from -1 to 1

Physical Interpretation: Projection of one vector onto another, normalized by their magnitudes

🧮 Vector Space Properties

📐 Vector Addition Properties:

  • Commutative: A + B = B + A
  • Associative: (A + B) + C = A + (B + C)
  • Zero Vector: A + 0 = A
  • Inverse: A + (-A) = 0

⚖️ Scalar Multiplication Properties:

  • Distributive: k(A + B) = kA + kB
  • Associative: k(mA) = (km)A
  • Identity: 1·A = A
  • Zero: 0·A = 0

🎯 Mathematical Applications

🔧 Physics & Engineering:

  • Force Vectors: F = ma, combining multiple forces
  • Velocity: v = displacement/time, velocity addition
  • Work: W = F·d (dot product of force and displacement)
  • Electric Field: E = kQ/r² direction vectors

📊 Simple Data Examples:

  • 2D Point: Position vector [x, y] represents location
  • Grade Vector: [Math, Science] = [85, 92] represents student performance
  • RGB Color: [Red, Green, Blue] = [255, 128, 64] represents color

🔬 Mathematical Foundation & Definitions:

  • Vector: Ordered n-tuple of real numbers: v = (v₁, v₂, ..., vₙ)
  • Magnitude: |v| = √(v₁² + v₂² + ... + vₙ²) - Euclidean norm
  • Unit Vector: ū = v/|v|, where |ū| = 1
  • Dot Product: A·B = Σ(aᵢbᵢ) = |A||B|cos(θ)
  • Vector Space: Set closed under vector addition and scalar multiplication
  • Linear Independence: No vector can be written as linear combination of others
  • Basis: Set of linearly independent vectors that span the space
  • Orthogonality: A ⊥ B ⟺ A·B = 0

⚡ Mathematical Exercises:

  1. Orthogonal Test: Set A = [3, 4], B = [4, -3] → A·B = 0
  2. Scaling: Scale A by 2 → |2A| = 2|A| (magnitude scales linearly)
  3. Basis Vectors: Try A = [1, 0], B = [0, 1] → Standard basis {e₁, e₂}
  4. Parallel Vectors: A = [2, 3], B = [4, 6] → cos(θ) = 1
