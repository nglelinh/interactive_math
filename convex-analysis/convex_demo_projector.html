üéØ What is a Convex Function?

A function $f(x)$ is convex if for any two points $x_1, x_2$ and any $\lambda \in [0,1]$:

$$f(\lambda x_1 + (1-\lambda)x_2) \leq \lambda f(x_1) + (1-\lambda)f(x_2)$$

üöÄ Key Insight

Any local minimum is a global minimum!

This makes optimization much easier and guarantees we'll find the best solution.

üîç Visual Test

Line Segment Test

Draw a line between any two points on the function. If the line segment lies above the function curve, it's convex.

üßÆ Mathematical Test

Second Derivative Test

If $f''(x) \geq 0$ for all $x$ in the domain, then $f(x)$ is convex.

üìä Function Examples

‚úÖ Convex Functions
  • Quadratic: $f(x) = ax^2 + bx + c$ (when $a > 0$)
  • Exponential: $f(x) = e^x$ (always convex)
  • Absolute Value: $f(x) = |x|$
  • Logarithmic: $f(x) = -\log(-x)$ for $x < 0$
‚ùå Non-Convex Functions
  • Sine: $f(x) = \sin(x)$ (oscillating)
  • Cubic: $f(x) = x^3$ (S-shaped)
  • Negative Quadratic: $f(x) = -x^2$ (upside-down parabola)

üéØ Why Convexity Matters

üöÄ Optimization Guarantees

Gradient descent will always find the global minimum, no matter where you start!

üß† Machine Learning Applications

Linear regression, logistic regression, and SVM all use convex loss functions for guaranteed optimal solutions.
